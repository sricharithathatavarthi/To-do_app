# 📝 Task Master – To‑Do List App

A simple yet powerful **Android To‑Do List application** built using **Android Studio**.  
This app helps users organize tasks, set priorities, and manage their schedules effectively.

---

## 🚀 Features
- ➕ **Add Tasks** – Create new tasks with titles and details.  
- 🎯 **Priority Levels** – Assign tasks as **Low, Medium, or High** priority.  
- 🔍 **Search & Filter** – Quickly find tasks using search and filter options.  
- ✅ **Mark as Completed** – Track progress by marking tasks done.  
- 📅 **Calendar Scheduling** – Plan tasks by date and time.  
- 📱 **Responsive UI** – Optimized for different screen sizes.  

---

## 🛠️ Tech Stack
- **Language:** Java  
- **IDE:** Android Studio  
- **Architecture:** MVVM (Model–View–ViewModel)  
- **Database:** Room / Firebase (depending on setup)  
- **UI Components:** RecyclerView, CardView, CalendarView
